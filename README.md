# SongForYou

A custom song ordering website connected to Stripe.

## Setup

1. Upload all files to GitHub
2. Connect GitHub repo to Vercel
3. Deploy!

## Your Stripe Link

Your payment link is already configured:
`https://buy.stripe.com/3cI28sd6l7Ks3IQ1HF53O01`

## How Orders Work

1. Customer fills out the song form
2. They copy their song details
3. They pay via Stripe
4. You receive payment notification
5. You create the song using Suno.ai
6. You email them the MP3

## Customization

Edit `src/App.jsx` to change:
- Price (search for "$19")
- Brand name
- Colors
- Form fields
